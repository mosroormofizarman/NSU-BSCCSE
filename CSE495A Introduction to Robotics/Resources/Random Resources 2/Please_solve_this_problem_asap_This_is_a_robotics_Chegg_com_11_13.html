Step 1/3
Sure I will provide you with the MATLAB code to calculate the output image G for each filter.

For filter F(a):


I = [7 4 1 ; 8 5 2 ; 9 6 3];
F = [0 0 0 ; 0 1 0 ; 0 0 0];
[m, n] = size(I);
[k, l] = size(F);
I_padded = padarray(I, [k-1, l-1], 0, 'both');
G = zeroes(m, n);

for i = 1:m
for j = 1:n
for u = 1:k
for v = 1:l
G(i, j) = G(i, j) + F(u, v) * I_padded(i+u-1, j+v-1);
end
end
end
end

disp(G);

Final answer
The filter in part (c) is a Sobel filter for edge detection. It calculates the horizontal gradient of the image, highlighting vertical edges.

The difference in functionality between filter F and F' is that F detects horizontal edges, while F' detects vertical edges in the image. F is a simple box filter, F' is a Sobil filter.
